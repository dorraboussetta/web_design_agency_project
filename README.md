# Creative Design Agency Website

## Introduction

This project is a responsive landing page for a fictional creative design agency. It was built to practice styling an HTML page with CSS.

## Project Overview

The website includes a logo, a main heading, two service cards, images, text content, and a footer. The layout changes based on screen size. On smaller screens, the cards stack vertically. On larger screens, the cards are displayed side by side.

## Technologies Used

* HTML5
* CSS3

## New Concepts Practiced

* CSS media queries
* Responsive design
* CSS floats
* Clearing floated elements
* CSS specificity
* CSS inheritence and specificity rules
* Image sizing with CSS
* Using `object-fit`
